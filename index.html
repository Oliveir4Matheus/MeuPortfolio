Projetos

  • Sistema de Gestão de Overtime

    Aplicativo desenvolvido para gestão eficiente do overtime da base. O sistema permite ao OCC informar horas extras autorizadas, enquanto a supervisão registra colaboradores em extra e períodos. O cruzamento de dados identifica autorizações, motivos, responsáveis e casos de trabalho sem autorização prévia.

    • HTML
    • CSS
    • JavaScript

  • Automações para Tratativas do Ponto

    Desenvolvimento de scripts Python para otimizar o alto volume de tratativas de ponto. Os principais scripts incluem:

    • Relatórios de inconsistências no ponto dos colaboradores
    • Relatórios de logs de acertos no ponto
    • Envio em massa de alteração de folgas e horários no sistema de otimização
    • Obtenção e inclusão de afastamentos em massa no sistema
    • Python
    • Automação
    • APIs

  • Sistema de Gestão de Handling (Malha V3)

    Este aplicativo foi desenvolvido com o objetivo de substituir a planilha de malha de voos em Google Sheets anteriormente utilizada pelo OCC, solucionando problemas como instabilidade na planilha, erros no preenchimento dos dados e ausência de informações essenciais.

    Por meio deste sistema, o OCC é capaz de:

    • Vincular os voos de chegada e saída
    • Escalar as equipes responsáveis por cada voo
    • Registrar informações operacionais (horário de pouso, calço, início da escala, limpeza e pushback)
    • Atualizar automaticamente um banco de dados diário com todas as informações da malha preenchida

    * Projeto desenvolvido em conjunto com outro colaborador da Swissport

    • React
    • Next.js
    • TypeScript
